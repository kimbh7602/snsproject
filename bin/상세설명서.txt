-메인페이지-

1. 마우스를 올리면 상품들의 이름과 상품정보로 이동하는 버튼을 보여줍니다.

2. 상단의 회원가입버튼을 누르면 회원가입 페이지로 이동합니다.
 
3. 회원가입에 성공하면 메인화면으로 이동합니다.
3. 로그인에 성공하면 상단에 자신의 랜덤 추천메뉴(오늘 뭐먹지?)와 자신의 ID, Logout버튼을 보여줍니다.

4. 상단에 '오늘 뭐먹지?' 를 누르면 랜덤함수를 사용한 알고리즘을 적용하여 상품 정보를 모달로 띄워줍니다.
(상품 정보 버튼 클릭시 해당 상품의 제품 정보 화면으로 이동합니다.)
5. 아이디를 누르면 회원정보 수정이 가능합니다.
6. 로그아웃시 메인화면으로 돌아갑니다.



-식품정보 관리-
7. 상품 정보 화면에서 상품명 검색시 키워드가 포함된 모든 상품을 출력해줍니다. (상품명, 제조사, 재료 모두 가능)

8. 셀렉트박스 옵션이 바뀔 시 상품명, 제조사에 대해 sorting알고리즘을 사용하여 리스트 출력하여줍니다.

9. 식품 검색시 첫글자를 입력하면 목록을 미리보기 해줍니다.
상품목록의 상품 이름을 클릭하면 해당 상품의 제품 정보 화면으로 이동합니다.

-제품 정보-
10. 상품 정보 화면에서 상품을 클릭하면 식품 상세정보 화면이 나타나고 제품명, 제조사, 원재료, 검색 빈도수를 나타내고 영양정보를 그래프로 나타냅니다.
11. 추가버튼을 클릭하면 나의 섭취 리스트에 추가되고, 섭취 상품 정보에서 확인 할 수 있습니다. 

-섭취 식품 정보-
12. 섭취 식품 정보 버튼을 누르면 사용자가 추가 버튼을 누른 전체 식품들과 총섭취 데이터 그래프를 보여줍니다.




13. 나의 섭취 리스트에 있는 상픔을 검색 할 수 있습니다. 

-게시판(질문게시판,공지게시판)
-- 게시판
14. 질문 게시판은 글 번호, 제목, 작성자 조회수를 나타내줍니다.
게시판은 로그인인 경우에만 글을 쓸 수 있습니다.
15. 게시판의 글을 누르면 수정,삭제,목록 버튼을 보여줍니다.
16. 질문 게시판 수정시, 원래 존재했던 글의 제목과 내용을 그대로 보여줍니다.
17. 수정, 삭제시 게시글의 작성자와 로그인된 아이디가 같을 시에만 수정, 삭제가 가능합니다.
--공지사항
18. 공지사항 게시판은 번호, 제목, 조회수를 나타내줍니다.
19. 공지사항 게시판은 관리자 계정만 글을 쓸 수 있습니다.
20. 셀렉트박스에 조회수순, 등록순, 최신순을  sorting 알고리즘을 사용하여 보여줍니다.
21. 공지게시판의 정렬은 option 선택으로 등록순, 조회수순, 최신순으로 가능합니다.

공지를 등록할 때마다 글 번호와 조회수 count를 증가시킵니다.
